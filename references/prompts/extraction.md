# AI 提取 Prompt 模板

你是一个信息提取专家。请从以下文章中提取结构化信息。

## 文章信息

**标题**：{{TITLE}}
**URL**：{{URL}}
**时间**：{{TIME}}
**内容**：
```
{{CONTENT}}
```

## 任务

### 1. 分类判断

从以下核心分类中选择**一个**最合适的分类：

{{CATEGORIES}}

如果文章内容涉及多个分类，选择最主要的一个。

### 2. 实体关系提取

提取文章中的实体关系（如果有）。关系类型包括：

- **投资**：投资方 → 被投公司（如：真格基金领投Manus A轮）
- **创始**：创始人 → 公司（如：肖弘创立蝴蝶效应）
- **产品**：产品名 → 公司（如：Manus是蝴蝶效应的产品）
- **工作**：人物 → 公司（如：张三曾任字节跳动技术VP）
- **收购**：收购方 → 被收购方（如：微软收购GitHub）
- **合作**：公司 ↔ 公司（如：OpenAI与Microsoft战略合作）

**提取原则**：
- 只提取文章中明确提到的关系，不要推测
- 每个关系包含：name（实体A）、relation（关系类型）、target（实体B）、note（补充说明）
- note 字段记录原文中的关键信息（如投资轮次、金额、职位等）

### 3. 生成摘要

用一句话（30字以内）概括文章的核心内容。

## 输出格式

请直接输出 JSON，不要有其他内容：

```json
{
  "category": "[分类名称]",
  "entities": [
    {
      "name": "实体A",
      "relation": "关系类型",
      "target": "实体B",
      "note": "补充说明"
    }
  ],
  "summary": "一句话摘要"
}
```

**注意**：
- 如果没有实体关系，entities 为空数组 `[]`
- category 必须从给定的分类列表中选择
- summary 必须简洁明了，不超过30字
