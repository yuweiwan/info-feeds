# 实体关系类型说明

本文档定义了 feeds skill 中支持的实体关系类型。

## 关系类型

### 投资 (investment)
- **方向**：投资方 → 被投公司
- **示例**：真格基金 → Manus（领投A轮）
- **提取要点**：投资轮次、金额（如有）、领投/跟投

### 创始 (founder)
- **方向**：创始人 → 公司
- **示例**：肖弘 → 蝴蝶效应（CEO）
- **提取要点**：职位、创始时间（如有）

### 产品 (product)
- **方向**：产品名 → 公司
- **示例**：Manus → 蝴蝶效应（AI Agent产品）
- **提取要点**：产品类型、发布时间（如有）

### 工作 (work)
- **方向**：人物 → 公司
- **示例**：张三 → 字节跳动（前技术VP）
- **提取要点**：职位、任职时间段（如有）

### 收购 (acquisition)
- **方向**：收购方 → 被收购方
- **示例**：微软 → GitHub（75亿美元）
- **提取要点**：收购金额（如有）、收购时间

### 合作 (partnership)
- **方向**：公司 ↔ 公司（双向）
- **示例**：OpenAI ↔ Microsoft（战略合作）
- **提取要点**：合作类型、合作内容

## 提取原则

1. **只提取明确的关系**：不要推测或臆断
2. **保留原文表述**：note 字段记录原文中的关键信息
3. **避免重复**：同一篇文章中相同的关系只记录一次
4. **优先级**：投资 > 创始 > 产品 > 工作 > 收购 > 合作
